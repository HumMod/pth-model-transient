Transient PTH Model

This mathematical model details the dynamics of PTH secretion in response to a fixed calcium function. It reduces to Brown's four parameter model (Brown, 1983) when constrained to steady state. Two versions of the code are presented; one in Mathematica, the other in the HumMod schema for use with DESolver.exe located on the HumMod GitHub site.

Mathematica Files

These files contain the fitting algorithms used to generate parameterizations for the transient PTH model. Data sets are used to both parameterize and validate the model; in particular, one interval of an experiment is used for fitting parameters, and the next interval of each experiment is used for testing the model predictions. The experimental PTH data is given in list form (Pexp and PexpLong) associated to a list of times at which measurements were taken (texp and texpLong). The -Long suffix is the full list, while the non-suffixed list is used for parameterization. The fits are performed by minimizing a least squares function over the parameter space; this format was used due to the individual nature of the tests involved.

HumMod files

A single model file, ssPTH.DES is currently included in this folder. The model shows the steady state predictions of the transient model, and allows a "hands-on" consideration of the sensitivity of the model on the parameters. The file should be opened in Model Editor.exe, also included. For the purpose of solving the model, the file DESolver.exe is also included. To run ssPTH.DES, open it in Model Editor.exe. Right click anywhere in the model text and select "Make this model root". In the "Options" tab, click "Controller-Solver" and select "Combined controller and solver". Browse to select DESolver.exe as the Controller-Solver and click Apply. Run the model from the tabs in the main Model Editor viewpane.
